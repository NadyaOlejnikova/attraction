## Аттракцион ##
+ Создана очередь на аттракцион и заполнена людьми из списка.
+ Заведен бесконечный цикл, который делает итерации пока очередь не пуста. На каждой итерации вытаскивается из очереди следующий клиент, выводится на экран сообщения. При этом у клиента уменьшается количество билетов на 1 и, если оно не стало нулевым, он вставляется обратно в очередь в её конец.
